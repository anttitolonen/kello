# Kello-projekti
Kello-projektin kunnianhimoisena tavoitteena on toteuttaa ohjelmistoratkaisu joka toimii koko elämän mittaisena ajanhallinnan partnerina kaikilla inhimillisen toiminnan alueilla.
## Asennus
Lataamalla julkaisusivustolta (https://github.com/anttitolonen/kello/releases) viimeisimmän version index.html-tiedostosta koneellesi ja avaamalla sen verkkoselaimella saapuu Kello elämääsi eikä poistu ennen kuin aika sinusta jättää.
